Contrôle d'Accès "Forteresse" (si vous trouvez la clé... ou le code... ou si la batterie n'est pas à plat)

(3.0 étoiles - Impénétrable... même pour le propriétaire)
99,98€ Niveau de Sécurité : "Bonne Chance !"

Piles non incluses (et elles durent ce qu'elles durent... c'est-à-dire pas longtemps)

Nul ne passera ! Sauf si vous oubliez le code. Ou si la batterie est à plat. Ou si l'écureuil du quartier a appris à taper "1234". Transformez votre porte en un défi quotidien !

  • Clavier Numérique Robuste : Touches garanties pour résister à au moins 100 pressions (après, c'est la loterie).
  • Codes d'Accès Multiples : Supporte jusqu'à 1 code utilisateur (celui que tout le monde oublie).
  • Verrouillage Automatique : Se verrouille après 3 tentatives erronées... ou parfois sans raison, pour garder le suspense.
  • Alarme Anti-Sabotage : Émet un "bip" discret si quelqu'un essaie de l'arracher du mur (ou si une mouche se pose dessus).
  • Clé d'Urgence Physique : Une petite clé est fournie. Ne la perdez pas. Sérieusement. C'est votre seul espoir si vous oubliez le code ET que la batterie est à plat.
  • Indicateur de Batterie Faible : Une LED rouge qui clignote... quand il est déjà trop tard.
  • Résistance aux Intempéries : "Relative". Évitez la pluie, la neige, le soleil direct, le brouillard, et les regards insistants.

Installation : Nécessite des compétences en perçage, câblage (beaucoup de fils de couleurs différentes, bonne chance !), et une patience de saint.

Configuration du Code : Le manuel (2 pages, écrites en Comic Sans par Kevin) explique comment changer le code par défaut ("0000"). Si vous y arrivez du premier coup, vous gagnez un cookie (virtuel).

Conseil de Pro (de Kevin) : Notez votre code sur un post-it et collez-le à côté du clavier. C'est plus simple.

Le Contrôle d'Accès "Forteresse" SécuriPLUS+ : Votre porte deviendra une énigme !

Transformez votre humble demeure en un bastion impénétrable (en théorie) avec notre système de Contrôle d'Accès "Forteresse". Fini les clés perdues ! Bonjour les codes oubliés et les batteries à plat au pire moment !

Ce système révolutionnaire (pour les années 80) utilise une technologie de pointe (un clavier numérique et quelques fils) pour vous offrir une tranquillité d'esprit... intermittente. Son design "brutaliste chic" (un gros boîtier en plastique) s'adaptera à tous les styles de portes, surtout celles qui ont déjà l'air un peu menaçantes.

Les avantages incontestables de la "Forteresse" :

  • Dissuasion : Les cambrioleurs verront le clavier et penseront "Trop compliqué, je vais chez le voisin".
  • Exercice Cérébral : Mémoriser le code (et où vous avez caché la clé d'urgence) stimulera vos neurones.
  • Conversation : "Alors, tu as réussi à rentrer chez toi hier soir ?" deviendra une question fréquente.
  • Le frisson de l'incertitude : La batterie va-t-elle tenir ? Ai-je le bon code ? Vais-je dormir dehors ce soir ? Tant de suspense !

Note de l'équipe de conception (Kevin, encore lui) : Le code "1234" est fortement déconseillé. Essayez "4321" pour plus d'originalité. Ou le code PIN de votre carte bancaire, comme ça vous ne l'oublierez pas (ne faites surtout pas ça).
